# خطوات تنفيذ المشروع على GitHub

## أولاً: إنشاء حساب
1. ادخل إلى موقع GitHub.
2. اضغط Sign up.
3. أنشئ حساباً باستخدام أي بريد إلكتروني لديك.
4. اختر اسم مستخدم واضح.
5. فعّل الحساب من رسالة التأكيد التي تصل إلى بريدك.

## ثانياً: فتح مشروع المعلمة
1. افتح رابط المشروع المرسل من المعلمة.
2. اقرأ التعليمات الموجودة في ملف README.

## ثالثاً: عمل نسخة من المشروع
1. اضغط زر Fork في أعلى الصفحة.
2. انتظر حتى يتم إنشاء نسخة من المشروع في حسابك.
3. تأكد أن اسم حسابك يظهر في رابط المشروع الجديد.

## رابعاً: حل الأسئلة
1. افتح الملف المطلوب مثل:
   - task1.py
   - task2.py
   - task3.py
2. اضغط على علامة القلم Edit.
3. اكتب الحل في المكان المحدد.
4. لا تغيّر أسماء الملفات.

## خامساً: حفظ الحل
1. بعد الانتهاء انزل إلى أسفل الصفحة.
2. اضغط Commit changes.
3. اكتب وصفاً بسيطاً مثل:
   - حل السؤال الأول
   - تعديل task1
4. اضغط Commit changes مرة أخرى إن ظهرت.

## سادساً: تسليم المشروع
1. افتح الصفحة الرئيسية لمشروعك بعد التعديل.
2. انسخ الرابط من أعلى المتصفح.
3. أرسل الرابط للمعلمة.

## ملاحظات مهمة
- يجب أن يكون الحل من عملك.
- لا تنسَ الضغط على Commit changes بعد كل تعديل.
- تأكد أن الرابط الذي ترسله هو رابط نسختك أنت، وليس رابط مشروع المعلمة.
